# Screenshots

Drop product screenshots here using these exact filenames — `index.html` already references them:

- `visual-tt.png`
- `storybuilder.png`
- `sfl-tracker.png`
- `ipad-dashboard.png`
- `lc-tracker.png`

**Format guidance:**

- **Aspect ratio:** 16:10 works best (matches the placeholder tile). Anything close is fine — CSS `object-fit: cover` will crop the rest.
- **Size:** 1600×1000 px (2× the rendered size on most screens). PNG or WebP.
- **File size:** under ~400 KB each. If a PNG is larger, run it through [TinyPNG](https://tinypng.com) or export as WebP.
- **Content:** real UI shots, not mocked frames. Schools-audience honesty is the brand.

**Placeholder behaviour:** while an image file is missing, the site shows a muted tile with the product name — the page still looks clean. When you drop a PNG in with the right filename, it replaces the placeholder automatically on next deploy.
